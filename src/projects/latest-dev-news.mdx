##### Backend
- AWS Lambda function written in Javascript.
- Deployed with Serverless Framework.
- Runs a cron job to get the most recent news.
- API is behind a CDN for caching.

##### Frontend
- Made with React.
- Styled with Tailwindcss.
- Deployed with Netlify.
